Dropbox Setup Guide

Do this once — then the app handles everything automatically forever.

1
Create a Dropbox App
Go to dropbox.com/developers/apps → Create app → Scoped access → App folder → any name (e.g. "LondonTrip") → Create app.
2
Leave Redirect URIs blank ← important!
On the Settings tab, look for the OAuth 2 Redirect URIs section — make sure it is completely empty. If there's anything listed there, delete it. This allows the auth code to appear on screen.
3
Set permissions
Go to the Permissions tab → tick files.content.read and files.content.write → click Submit.
4
Copy your App Key and App Secret
Go to the Settings tab. You'll see App key and App secret — paste both into the fields, then tap Authorise with Dropbox. The rest is automatic.
